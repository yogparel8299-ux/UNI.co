# UNIC.ai

UNIC.ai is a light-premium AI Agent Operating System.

## Included

- Landing page
- Dashboard
- AI Command Center
- Companies
- Agents
- Swarms
- Tasks
- Builder
- Datasets
- Marketplace
- Approvals
- Usage
- Activity
- Billing
- Schedules
- Settings
- Legal pages
- Supabase admin client
- AI command planner
- Queue API
- Command API
- DigitalOcean worker
- OpenAI execution support
- Usage event tracking
- Runtime event tracking

## Commands

npm install
npm run dev
npm run build
npm run worker

## Required environment variables

NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY
SUPABASE_SERVICE_ROLE_KEY
OPENAI_API_KEY
NEXT_PUBLIC_APP_NAME
NEXT_PUBLIC_APP_URL
UNIC_WORKER_SECRET
